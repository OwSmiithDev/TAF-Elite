# Supabase Edge Functions

This directory contains Supabase Edge Functions.

## How to deploy

1. Install the Supabase CLI:
   ```bash
   npm install -g supabase
   ```

2. Login to Supabase:
   ```bash
   supabase login
   ```

3. Link your project:
   ```bash
   supabase link --project-ref your-project-ref
   ```

4. Deploy a function:
   ```bash
   supabase functions deploy hello-world
   ```
